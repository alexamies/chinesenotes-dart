Glossary of the Lotus Sūtra (Dharmarakṣa)

This extension is based on A Glossary of Dharmarakṣa's Translation of the Lotus Sūtra by Seishi Karashima (1998). Use the dicitonary either by selected text on the page and right clicking to use the context menu or from this popup page. Search for a term. Examples: 阿難律, 阿須倫. Open this page in a new tab to avoid needing to reload it each time.

Buddhist Multi-Dictionary Workbench

This extension is based on a combination of NTI Reader, FGS HB Glossary, Mahavyutpatti, Soothill-Hodous, Glosaries of the Aṣṭa, Dīrgha-āgama, Lotus Sūtra, and DDBC Authority Database. Copyright by various. Use the dicitonary either by selected text on the page and right clicking to use the context menu or from this popup page. Search for a term. Examples: 佛, 如來, 不動, 鐘頭, 一切智, 定, 和, 布施, 五蘊, 無著 (Asaṅga), or 平城京 (Heijō-kyō). Also try reverse lookup with an English, Sanskrit (pañcaskandha), Pali (pañcakhandhā), Japanese (goun), or Tibetan equivalent. Open this page in a new tab to avoid needing to reload it each time.

DDBC Person and Place Authority Database

This extension is based on the Person and Place Authority Databases form the Library and Information Center at Dharma Drum Institute of Liberal Arts. Use the dicitonary either by selected text on the page and right clicking to use the context menu or from this popup page. Search for a term. Examples: 無著 (Asaṅga), 平城京 (Heijō-kyō). Open this page in a new tab to avoid needing to reload it each time.
